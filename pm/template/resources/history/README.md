# history/

Milestone records — significant one-offs that happened and are worth remembering after the fact.

Things that live here:

- Scaffold notes (`2026-04-10-initial-scaffold.md` — what was set up, what was chosen, what was deferred)
- Significant migrations (data, framework upgrades, domain moves)
- Engagement-level retrospectives (separate from per-milestone retros in `../../_pm/milestones/`)
- Incident write-ups when a production issue produced learnings worth keeping

**Filename convention:** `YYYY-MM-DD-<topic>.md`.

Day-to-day work lives in `../../_pm/sessions/`. Per-milestone narrative lives in `../../_pm/milestones/M*/retro.md`. This folder is for the *project-level* events that span multiple sessions or milestones.
